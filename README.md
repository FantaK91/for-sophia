# For Sophia - carta interactiva móvil

Proyecto listo para GitHub Pages.

## Archivos
- `index.html`
- `style.css`
- `script.js`
- imágenes y video en la raíz

## Publicación
1. Sube todos los archivos al repositorio.
2. En GitHub: **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main`
5. Folder: `/(root)`
6. Guarda y espera.

## Nota
Este paquete evita la carpeta `assets` para que no fallen las rutas al subirlo a GitHub.
